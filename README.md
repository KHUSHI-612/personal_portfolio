# Personal Portfolio

A clean and responsive personal portfolio website built with React and Vite.

## Tech Stack

- React
- Vite
- CSS (Vanilla)
- Lucide React (Icons)
- React Scroll

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
